📋 活动报名签到 Demo

Event Registration & Check-in — 纯前端交互示意

🎯 功能说明

这是一个活动报名与签到系统的产品雏形。后端使用 Python 标准库实现，前端 HTML 页面展示交互流程。

创建活动 — 后端 Event(id, title, capacity) 创建活动，设定容量上限。

用户报名 — 通过 event.register(email, name) 报名，校验邮箱格式、去重、容量检查。

现场签到 — 通过 event.check_in(email) 签到，校验是否已报名、是否重复签到。

数据持久化 — EventStore.save_to_file() / load_from_file() 支持 JSON 文件读写。
